Projects

Dual Gripper System

RAMP Lab - Simon Fraser University

Burnaby, Canada

The Dual Gripping System integrates parallel-jaw and suction grasping capabilities into a single adaptive robotic gripper. Designed for versatility, the system employs deep learning to optimize grasping strategies for a variety of objects and environments. By combining pinch and suction grasps, the gripper ensures enhanced stability and adaptability in handling tasks, paving the way for innovative applications in automation and robotics.

Skill utilized: Machine Learning, Python, Linear Programming, TensorFlow, Robotics, Computer Vision, Mechanical Design, Simulation, Hardware Prototyping, and Technical Documentation.

DualGripNet

RAMP Lab - Simon Fraser University

Burnaby, Canada

DualGripNet is a deep learning model developed for a dual robotic gripper capable of performing pinch and suction grasps. Synthetic data was generated in simulation by sampling diverse object shapes and annotating them with potential pinch and suction grasp points. The model was trained on this dataset to predict grasp types and locations based on object geometry. DualGripNet enables the gripper to choose and execute the most suitable grasp for stable object handling.

Skills utilized: Deep Learning, Synthetic Data Generation, Robotics, Grasping, Data Processing and Annotation, Python, TensorFlow.

Phoenix Robot Testing with BehaviourTree Framework

Software Team - Advanced Intelligent Systems (AIS)

Burnaby, Canada

Developed a robust testing framework for the Phoenix robot using BehaviorTree.CPP and Groot. Designed a ROS-based environment with predefined modules such as low_battery, collision, and delay, enabling seamless creation of test cases. The solution included an intuitive interface via Groot, allowing non-technical users to connect test nodes and validate various robotic functionalities efficiently. This framework streamlined the testing process and improved collaboration across teams.

Skills utilized: ROS, Robot Testing, BehaviourTree, C++, Software Integration

Optimal Gait Design for Lower Extremity Exoskeleton (Exoped)

Pedasys Company

Tehran, Iran

In this project, we proposed a systematic approach to design an optimal gait for lower extremity exoskeletons, addressing the stability challenges faced by users. A feedback-controlled system was developed to dynamically adjust walking parameters, ensuring smooth motion. By solving an optimal control problem and minimizing cost functions, the method generates walking patterns with spline interpolation and sinusoidal trajectories. The proposed approach enhances stability, adaptability, and user comfort in robotic assistive devices, validated through simulations and analysis.

Skill utilized: Robotics, Control Systems, Optimal Control, Feedback Control System, Optimization, MATLAB.

Fractrional order Multi-agent system Consensus

Tarbiat Modares University

Tehran, Iran

This research addresses the consensus problem in nonlinear fractional-order multi-agent systems (FOMAS) with uncertainties. It introduces a dynamic output feedback controller leveraging neighboring agents’ outputs to ensure robust stability using Lyapunov methods and linear matrix inequalities (LMIs). The study accounts for nonlinearities and uncertainties through advanced mathematical frameworks like the S-procedure and Schur complement. Simulation results validate the proposed method’s efficiency in achieving consensus under challenging conditions, demonstrating its applicability in control engineering.

Skill utilized: Control Systems, Linear Programming, Optimization, MATLAB.
